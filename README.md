
# Universal Bank AI Marketing Dashboard

This project predicts which customers are likely to accept a personal loan.

## Features
• Descriptive analytics of customer data
• ML models: Decision Tree, Random Forest, Gradient Boosting
• Model comparison table
• ROC curve comparison
• Confusion matrices
• Feature importance insights
• Upload customer data to predict loan acceptance

## Run locally
pip install -r requirements.txt
streamlit run app.py

## Deploy
Upload to GitHub and connect repository on Streamlit Cloud.
